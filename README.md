Skype Tools
===========

## これはなに
Skypeのmain.dbを読んで、いろんな情報を解析するツール(らしい)

## 必要なもの
* main.db (オリジナルをコピーして使うことを推奨します)
    * Windows: C:\Users\<Windowsのユーザー名>\AppData\Roaming\Skype\<SkypeName>\main.db
    * Linux: ~/.Skype/<SkypeName>/main.db

## 含まれているスクリプト
* search.rb (検索)
* count.rb (発言数カウント)
